# MiniMax语音 (Pro) - SillyTavern  TTS 扩展

**MiniMax语音** 是一款专为 SillyTavern 设计的深度定制 TTS 扩展。它不仅支持高质量的 MiniMax 语音合成，还引入了“副 API 格式化”引擎，能够通过 LLM 智能识别文本中的说话人、情感和语气，为你的角色注入真正的灵魂。

为什么要做这样一款插件？酒馆原生的TTS比较呆板，很多功能也缺失。同时，不是所有的人都有好的显卡电脑可以跑本地TTS，这个时候MiniMax的API语音服务就是一个比较好的选择。

## ✨ 核心特性

- **🚀 深度集成 MiniMax V2**：支持 `speech-02-hd` 等最新模型，具备极高的拟合度和流畅感。
- **🤖 副 API 智能格式化**：可调用外部 LLM 预处理文本，自动分段并标注说话人与情绪（如：开心、悲伤、愤怒），可根据需求拟定提示词，灵活好上手。
- **🎭 角色感应绑定**：绑定配置随角色卡走。换人自动切配置，支持为每个角色（及用户），包括自定义NPC设定独立的音色和模型。
- **🔄 多版本 Roll 声音系统**：
  - **长按**：Roll 一个新版本（重新请求 LLM 标注情感或重新合成）。
  - **双击**：进入历史管理面板，左右切换所有 Roll 过的版本，对比选择最满意的那个。
- **💾 云端持久化缓存**：音频自动备份至酒馆服务器。换浏览器、换电脑，只要登录同一账号，之前的声音无需重复消耗 API 额度即可秒速播放。

## 💡 交互逻辑

为了平衡自动化与精细控制，本插件采用以下操作逻辑：

| 操作 | 逻辑说明 |
| :--- | :--- |
| **单击** | **播放**当前选中的版本。如果开启了副 API 且未 Roll 过，会提示先长按。 |
| **长按 (600ms)** | **生成/重 Roll**。强制副 API 重新分析文本结构并生成全新版本。 |
| **双击** | **微调面板**。左右切换历史版本，修改说话人、语音 ID、语速、音量等参数。 |

## ⚙️ 配置指南

### 1. 基础配置
在扩展面板填入你的 MiniMax `API Key` 和 `Group ID`。点击 **“测试 MiniMax 语音”**，听到“你好”即表示连接成功。

### 2. 副 API 格式化 (建议开启)
- **接口格式**：支持 OpenAI 兼容格式
- **提示词**：支持多个提示词模板预设保存

建议在“提示词”框中使用如下逻辑：
> 你是语音标注专家。请将输入的文本转化为严格的 JSON 格式：
> `{"segments":[{"text":"文本内容","speaker":"说话人姓名","emotion":"happy/sad/neutral","speed":1.0}]}`
> 只保留可朗读内容，不要包含 ```json 等标记。

可以根据不同的需求撰写不同提示词，例如需要旁白等。

### 3. 角色绑定
根据 LLM 识别出的“说话人”名字，在这里添加对应的语音 ID。例如：
- 识别出 `Char` -> 绑定 `male-qn-qingse` 。
相应的char就可以用相应的语音ID声音说话，不同的角色声音也不同

---

### 4. Minimax获得简单教程

1. 去往minimax中国大陆官网用手机号注册即可，新注册用户送10块钱，能用挺久一段时间
2. 得到Group ID，创建API Key
3. 去往minimax国际版官网，用外网账号（例如谷歌号）注册登录，克隆你想要的声音，得到语音ID，每个账号可以免费克隆3次

## 🛠️ 安装方法

1. **下载插件**：在扩展页面中粘贴github链接下载：https://github.com/dream-ice/minimax-quote-tts.git
2. **替换后端核心 (关键)**：为了解锁情感控制和多域名切换，请用下面的 `minimax.js`文件替换掉酒馆源码中的 `src/endpoints/minimax.js`。
3. **重启酒馆**：彻底关闭并重新启动酒馆后端程序。
